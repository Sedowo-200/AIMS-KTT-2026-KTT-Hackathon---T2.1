## Docker Instructions

This project exposes the INT8 crop disease classifier through a FastAPI `/predict` endpoint.

### Prerequisites

Install Docker Desktop, then verify Docker is available:

```bash
docker --version
```

Make sure `model.tflite` exists at the project root before building the image.

### Build

Run this command from the project root:

```bash
docker build -t crop-classifier -f service/Dockerfile .
```

### Run

Start the API container:

```bash
docker run --rm -p 8000:8000 crop-classifier
```

### Health Check

Check that the service is running:

```bash
curl http://127.0.0.1:8000/health
```

### Prediction Example

Send one test image to the `/predict` endpoint:

```bash
curl -X POST "http://127.0.0.1:8000/predict" \
  -F "file=@data/mini_plant_set/test/maize_rust/maize_rust_0001.jpg"
```

Expected response format:

```json
{
  "label": "maize_rust",
  "confidence": 0.89,
  "top3": [],
  "latency_ms": 45,
  "rationale": "Prediction generated by an INT8 MobileNetV3-Small model from the uploaded leaf image."
}
```
